# Health Insurance Fraud Detection

## Project Overview

This project aims to detect health insurance fraud using machine learning techniques. We used Medicare claims data to build predictive and anomaly detection models.

## Project Structure

- `data/`: Raw datasets
- `notebooks/`: Jupyter notebooks for preprocessing, modeling, anomaly detection
- `scripts/`: Reusable Python scripts
- `outputs/`: Saved charts, confusion matrix images

## Methodology

- Data merging and preprocessing
- Feature engineering
- SMOTE balancing
- Model building: Random Forest, Logistic Regression, ANN
- Anomaly detection: Isolation Forest, One-Class SVM

## Evaluation Metrics

- Accuracy, Precision, Recall, Specificity, F1 Score, AUC Curve

## Unique Insights

- Claims frequency patterns
- Chronic condition vs hospitalization mismatch

## How to Run

1. Clone the repo
2. Install dependencies using `pip install -r requirements.txt`
3. Run notebooks step-by-step.
